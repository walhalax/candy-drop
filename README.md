# ぷよぷよ

Webブラウザでプレイできる классир ぷよぷよゲーム。

## 遊び方

### 操作方法
| キー | 動作 |
|------|------|
| ← | 左移動 |
| → | 右移動 |
| ↓ | 落下加速 |
| . (ピリオド) | 左回転 |
| / (スラッシュ) | 右回転 |
| Space | 停止 / 再開 |

### ルール
- 同じ色のぷよが **4つ以上繋がる** と消去
- 連鎖すと加点
- 積み上がるとおじゃまぷよが落ちてきて、ゲームオーバー

## 起動方法

```bash
open index.html
```

またはブラウザで `index.html` を開く。

## 技術仕様
- 单一HTML + Vanilla JS + Canvas API
- 外部依存ナシ
