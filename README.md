# Candy Drops

Webブラウザでプレイできる 落ちものパズルゲーム。

## 遊び方

### 操作方法（PC）
| キー | 動作 |
|------|------|
| ← | 左移動 |
| → | 右移動 |
| ↓ | 落下加速（長押し: 一気に落下） |
| . (ピリオド) | 左回転 |
| / (スラッシュ) | 右回転 |
| Space | 停止 / 再開 |

### 操作方法（バーチャルコントローラー）
- **左側**: ←→ 十字キーで移動、↓長押しでハードドロップ
- **右側**: ↺左回転 / ↻右回転
- 全デバイス（PC/iPad/iPhone）表示対応

### ルール
- 同じ色のキャンディが **4つ以上繋がる** と消去
- 連鎖すると加点
- ハードモードでおじゃまキャンディが落ちてくる
- 積み上がるとゲームオーバー

## 起動方法

```bash
open index.html
```

またはブラウザで `index.html` を開く。

## 技術仕様
- 单一HTML + Vanilla JS + Canvas API
- 外部依存ナシ
- PWA対応

## 主要機能

### マジックボタン
- 、画面下部に配置された特殊ボタン
- チャージ式で蓄積後に発動
- 1.25倍サイズで表示

### タイトル画面
- 難易度選択時に選択項目がきらきら光る装飾
- ゲームスタートボタン（0.8倍サイズ）

### メニュー
- ゲーム中にメインメニュー押下で確認ポップアップ表示
- キャンセルで即時ゲーム再開
