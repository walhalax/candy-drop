# Candy Drops 難易度拡張設計

## 概要
- 難易度選択システム追加
- iPadスワイプ慣性ドロップ実装
- 、透明ジャマブロック仕様

## 1. スワイプ慣性ドロップ

### 速度計算
- スワイプ距離 ÷ 時間 = 速度（pixels/ms）

### ドロップ幅マッピング
| 速度 (px/ms) | ドロップ幅 |
|--------------|-----------|
| < 0.5 | 1セル |
| 0.5 - 1.0 | 3セル |
| 1.0 - 2.0 | 6セル |
| > 2.0 | 9セル（最大） |

## 2. 難易度システム

### 定数定義
```javascript
const DIFFICULTY = {
  EASY: 'easy',      // ふつう
  HARD: 'hard',      // むずかしい
  EXTREME: 'extreme' // めちゃムズ
};
```

### 難易度別仕様
| 難易度 | 落下速度 | 特殊処理 |
|--------|----------|----------|
| ふつう | 800ms固定 | なし |
| むずかしい | 10ドロップごとに1% speedUp | 速度上昇 |
| めちゃムズ | 10ドロップごとに1% speedUp | +15ドロップごとに透明ブロック |

### 速度計算
```javascript
// むずかしい・めちゃムズ共通
const speedMultiplier = Math.pow(1.01, Math.floor(totalDrops / 10));
const currentFallInterval = BASE_FALL_INTERVAL / speedMultiplier;
```

## 3. 透明ブロック仕様

### 描画
- alpha = 0.25
- 色: #aaaaaa（薄い灰色）
- 布雷 доллтон 薄いボーダー

### 動作
- プレイヤー操作不能
- 自由落下（落下速度 = 通常ブロックの1.5倍）
- 他のブロックとの衝突判定なし

### 専用シーン（ちゃうブロック投入）
- トリガー: 15ドロップごとに1回
- 処理時間: 1.5秒
- 動作:
  1. 通常のブロック落下を停止
  2. 画面上部のランダム位置に透明ブロック1個生成
  3. 自由落下（最速）
  4. 床または最下層に達したら積る
  5. 1.5秒後に通常ブロック落下再開

### 消失条件
- 隣接するブロックが消失した時のみ
- 透明ブロック同士の隣接では消えない
- 消失時は通常の破裂アニメーション（alpha: 0.25）

## 4. ゲームオーバー条件

- 通常ブロック: 列最上部に達したらゲームオーバー
- 透明ブロックが積まれた場合: その列の通常ブロックもゲームオーバー判定

## 5. UI変更

### スタート画面
- 難易度選択ボタンを3つ追加
- 「ふつう」「むずかしい」「めちゃムズ」
- 選択時にvisual feedback
