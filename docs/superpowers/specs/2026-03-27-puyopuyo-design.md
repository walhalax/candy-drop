# ぷよぷよゲーム設計書

## プロジェクト概要
- **プロジェクト名**: ぷよぷよ（Web版）
- **タイプ**: 落ち物パズルゲーム
- **コア機能**: キーボード操作可能なClassicなぷよぷよ
- **ターゲットユーザー**: ぷよぷよファン、懐かしさを求めるプレイヤー

---

## ゲーム内容

### ルール
- Nexon/アージュ版と同じ **4個-connect（縦・横4つ以上繋がれば消去）**
- ぷよを落として、同じ色のぷよが4つ以上繋がると消去
- 連結したグループが落下して新しい連結が生まれると **連鎖** が発生

### 操作
| キー | 動作 |
|------|------|
| ← | 左移動 |
| → | 右移動 |
| ↓ | 落下加速 |
| . (ピリオド) | 左回転 |
| / (スラッシュ) | 右回転 |
| Space | 一時停止 |

### 上級玩法
- **回転入れ**対応（複雑な置き方で加点・連鎖機会創出）

### 続行条件
- **Perpetual形式**
- ぷよが積み上がり、天井に到達するとおじゃまぷよが落下
- おじゃまぷよが画面最下部に到達するとゲームオーバー

### スコア
- 連結・連鎖による得点制
- 連鎖数は1, 2, 3...とカウントアップ

## 回転ルール補足（2026-03-28 実装）

### 条件
- mainブロックの**真下に空間がある**（接地していない）
- 左右がブロックで挟まれている（90度/-90度回転不可）

### 動作
- 上記条件下で**物理swap回転**を実行
- mainとsatelliteの位置を入れ替え（色も跟着移動）

### 実装詳細
1. `getSatellitePos()`でsatelliteの現在位置を取得
2. mainの色とsatelliteのの色を交換
3. mainの位置をsatelliteの位置に移動
4. `isRotating = true` で回転アニメーション再生

### 全ケース整理
| 状態 | 動作 |
|------|------|
| main真下有空 + 左右挟まれ | **物理swap回転**（上下入れ替え） |
| main真下有空 + 左右空いている | 通常の90度/-90度回転 |
| main真下无空（接地） | 回転不可 |
